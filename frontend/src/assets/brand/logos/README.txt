KALLINGAL — ABOUT PAGE PARTNER LOGOS
====================================

Drop each partner-brand logo here as a PNG with a TRANSPARENT background.
The About page ("We offer products & services of") shows the image
automatically. Until a file exists, that tile shows the brand name instead.

File naming: lowercase, spaces -> hyphens, e.g. "JK Tyre" -> jk-tyre.png

Notes:
- Recommended: ~400px wide, transparent PNG. Logos display on a white tile,
  so dark/coloured logos read best.
- To use a different filename, an SVG, or an external URL, set the "Logo URL"
  field for that brand in Admin -> Site Content -> About-page brands.
- The grid auto-updates when new brands are added in the admin; just add a
  matching <name>.png here (or set its Logo URL).
